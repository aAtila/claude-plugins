# Atila's Claude Code Plugins

A collection of plugins for Claude Code.

## Installation

```bash
/plugin marketplace add aAtila/claude-plugins
```

Then install any plugin:

```bash
/plugin install seo@aAtila
/plugin install webdev@aAtila
```

Restart Claude Code after installing.

## Available Plugins

### seo (v0.2.0)

SEO best practices for web development.

**Skills:**
- `semantic-html` - Catches semantic HTML issues during development
- `semantic-review` - Comprehensive semantic audit for completed pages

### webdev (v0.1.0)

Web development workflows and best practices.

**Skills:**
- `commit` - Structured commit messages with conventional commits format
